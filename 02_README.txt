# Basic Blog Page

This is a simple HTML + CSS project that creates a basic blog website.

## Features:
- Header with blog title and tagline
- Two sample blog posts
- Clean layout with styled text
- Footer with copyright

## Files:
- 01_index.html → Main HTML file
- 01_style.css → CSS styling file

## How to Run:
1. Open the folder in VS Code.
2. Open index.html in a browser (Right-click → Open with Browser).
3. You will see the blog page.
