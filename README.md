# Simple Blog Website

A simple blog website built using HTML and CSS. This project demonstrates the fundamentals of webpage structure, content organization, and styling by creating a clean and readable blog layout.

## 📌 Project Overview

This project is designed to simulate a basic personal blog where users can publish and display blog posts in an organized format.

The website contains multiple blog posts, a header section, and a footer while maintaining a simple and responsive design structure.

It is a beginner-friendly project for practicing HTML and CSS concepts.

## 🚀 Features

* Clean and simple blog layout
* Multiple blog post sections
* Styled header and footer
* Readable typography
* Card-style blog articles
* Responsive content width
* Beginner-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
